# Netflix Data Analysis

## Overview
This project analyzes Netflix movies and TV shows dataset to uncover trends in content production, popular genres, and regional distribution.

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Insights
- Drama and International Movies are the most dominant genres
- Significant growth in content production after 2015
- United States contributes the highest number of titles
- Increasing trend of TV Shows in recent years

## Conclusion
The analysis highlights how Netflix has expanded its content library globally, with a strong focus on diverse genres and rapid growth in the past decade.
